Generate SSL CSR and Key Noninteractively


openssl req -new -nodes -sha256 -newkey rsa:2048 -keyout <key_name> -out <csr_name> -subj "/emailAddress=<email>/CN=<cname>/O=<organization>/OU=<org_unit>/C=<country>/ST=<province>/L=<locality>"

Country Name(2 letter code)

State or Province Name

Locality Name(city)

Organization Name(company)

Organizational Unit Name

Common Name(server FQDN)

Email Address
